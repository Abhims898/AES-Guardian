About AES Encryption

The Advanced Encryption Standard (AES) is a symmetric block cipher used by governments and organizations worldwide to protect sensitive data.

Key Features:

  • Symmetric algorithm (same key for encryption and decryption)
  • Block size: 128 bits (16 bytes)
  • Key sizes: 128, 192, or 256 bits
  • Secure against all known practical attacks

Key Handling: Any length key is automatically hashed to create secure AES keys (SHA-256 used for key derivation).
